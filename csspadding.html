CSS Padding Padding is the space between the HTML content and the border. It gives internal spacing and helps to enhance the website.




Method 1

Syntax


selector{

padding: value;

}

Example


<head>
<style>
#p1{
border: 2px solid purple;
}
#p2 {
padding: 12px;
border: 2px solid red;
}
</style>
</head>
<body>
<p id="p1">CodeWithme (without padding)</p>
<p id="p2">CodeWithme (with padding)</p>
</body>
Method 2

Syntax


selector{

padding: value1 value2;

}

Example


<html lang="en">
<head>
<style>
#p1{
border: 2px solid purple;
}
#p2 {
padding: 20px 50px;
border: 2px solid red;
}
</style>
</head>
<body>
<p id="p1">CodeWithme (without padding)</p>
<p id="p2">CodeWithme (with padding)</p>
</body>
</html>
Method 3

Syntax


selector{

padding: value1(top) value2(right) value3(bottom) value4(left);

}

Example

<html lang="en">
<head>
<style>
#p1{
border: 2px solid purple;
}
#p2 {
padding: 15px 30px 25px 50px;
border: 2px solid red;
}
</style>
</head>
<body>
<p id="p1">CodeWithme (without padding)</p>
<p id="p2">CodeWithme (with padding)</p>
</body>
</html>
